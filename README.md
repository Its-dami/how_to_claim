## 这是什么

一个对话式训练 Skill，专门训练有法律基础的初级法律人完成民事诉讼中最核心的工作：**选对诉讼请求、找准请求权基础、拆清债法关系、预判攻防、对齐证据。**

不做空泛讲解，不输出学术论文。每一轮训练都围绕一个案件，把"要法院判什么"这个问题一步一步拆到底。

## 适用场景

- 准备起诉，不确定诉讼请求怎么选
- 收到起诉状，需要设计答辩和抗辩策略
- 庭前推演，模拟被告最强抗辩和法官追问
- 证据梳理，按构成要件排列而非按时间线堆砌
- 训练请求权基础方法和债法关系拆解

## 不适用

- 刑事案件
- 行政诉讼
- 仲裁程序策略
- 执行程序
- 纯程序问题（管辖、保全、上诉期限等仅做辅助提示）

## 法律基准

默认适用中华人民共和国大陆地区现行法律：

- 《中华人民共和国民法典》（2021年1月1日施行）
- 现行最高人民法院司法解释
- 《中华人民共和国民事诉讼法》（2023年修正，2024年1月1日施行）
- 大陆民法学说与审判实务

参考书中涉及的台湾地区条文号、判例和制度结论，仅作为比较法和方法论材料，不直接用于训练答案。

## 训练流程

```
案件压缩 → 债法关系定位 → 诉讼请求候选 → 请求权基础匹配
    → 胜算排序 → 攻防推演（被告抗辩 / 原告反击 / 法官追问）
        → 证据缺口识别 → 复盘评分
```

## 训练模式

| 模式 | 适用场景 |
|------|----------|
| 原告进攻 | 起诉、写诉状、设计请求 |
| 被告防御 | 应诉、答辩、反驳诉请 |
| 债法关系定位 | 合同、侵权、不当得利等债因拆解 |
| 法官追问 | 庭审表达和裁判视角训练 |
| 快速诊断 | 只要策略方向，不展开推演 |

## 方法底盘

本 Skill 的训练方法由 yangyifang 基于以下方法论体系整理设计：

- **请求权基础方法**：谁得向谁，基于何种法律依据，主张何种法律救济
- **债法统一入口**：以《民法典》第118条为基础，所有训练都从"给付是什么"切入
- **三层检视**：请求权是否产生、是否消灭、是否可行使
- **义务群拆解**：主给付 / 从给付 / 附随义务 / 先契约义务 / 后契约义务
- **攻防映射**：原告证产生要件，被告攻要件、提抗辩、制造证明困难
- **证据对应**：每一份证据对应一个构成要件或抗辩事实

## 评分体系

每轮训练结束后，yangyifang 设计的六维评分：

| 维度 | 满分 |
